## 2. 路由设计

| 路径 | 页面 | 说明 |
|------|------|------|
| `/login` | 登录页 | 用户登录 |
| `/` | Dashboard | 数据看板（默认首页） |
| `/social/tasks` | 社媒任务列表 | 管理社媒挖掘任务 |
| `/social/tasks/create` | 创建社媒任务 | 表单页 |
| `/social/tasks/:id` | 任务详情 | 任务状态 + 关联线索 |
| `/social/leads` | 社媒线索列表 | 筛选、排序、导出 |
| `/social/leads/:id` | 线索详情 | 来源内容 + AI 分析 |
| `/b2b/tasks` | B2B 任务列表 | 管理 B2B 搜索任务 |
| `/b2b/tasks/create` | 创建 B2B 任务 | 表单页 |
| `/b2b/tasks/:id` | 任务详情 | 任务状态 + 关联线索 |
| `/b2b/leads` | B2B 线索列表 | 筛选、排序、导出 |
| `/b2b/leads/:id` | 线索详情 | 公司信息 + 联系人 |
| `/settings` | 系统设置 | AI 配置、API 用量 |

## 3. 页面设计

### 3.1 登录页

- 居中卡片式登录表单
- GlobalLeads Logo + 产品名
- 用户名 + 密码输入框
- 登录按钮

### 3.2 主布局

- **顶部导航栏 (Header)**
  - 左侧：GlobalLeads Logo
  - 右侧：用户头像 + 退出登录

- **侧边栏 (Sidebar)**
  - 数据看板
  - 社媒线索挖掘
    - 任务管理
    - 线索库
  - B2B 客户搜索
    - 任务管理
    - 线索库
  - 系统设置

## 4. 组件设计

### 4.1 通用组件

| 组件 | 说明 |
|------|------|
| `AppLayout` | 主布局（Header + Sidebar + Content） |
| `StatusTag` | 状态标签（pending=蓝/running=橙/completed=绿/failed=红） |
| `ScoreBar` | 意向评分进度条（0-100，颜色渐变） |
| `PlatformIcon` | 平台图标（Reddit/Bluesky/YouTube） |
| `ExportButton` | 导出按钮（支持 CSV/Excel） |

### 4.2 业务组件

| 组件 | 说明 |
|------|------|
| `SocialTaskForm` | 社媒任务创建/编辑表单 |
| `B2BTaskForm` | B2B 任务创建/编辑表单 |
| `SocialLeadTable` | 社媒线索表格（带筛选、排序、批量操作） |
| `B2BLeadTable` | B2B 线索表格 |
| `LeadDetailPanel` | 线索详情抽屉/面板 |
| `DashboardCharts` | 看板图表组 |

---

## 5. 状态管理

使用 React Hooks + Context 管理状态，不引入 Redux：

- `AuthContext` — 用户认证状态、Token 管理
- 各页面内部状态通过 `useState` + `useEffect` 管理
- API 调用通过 `services/` 层统一封装

---

## 6. 主题

延续 LeadMine 的 Ant Design 风格，中文界面：
- 主色调：蓝色系（#1677ff）
- 语言：中文
- 响应式：支持桌面端，移动端适配
