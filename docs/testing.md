## 4. 集成测试（端到端）

### 4.1 社媒线索挖掘完整流程

```
1. 创建社媒任务（关键词："LED lighting"，平台：Reddit）
2. 确认任务状态变为 running
3. 等待任务完成
4. 确认任务状态变为 completed
5. 查看 social_leads 表有新数据
6. 通过 API 获取线索列表
7. 确认线索包含正确的 AI 评分和标签
8. 导出 CSV 并验证内容
```

### 4.2 B2B 客户搜索完整流程

```
1. 创建 B2B 任务（行业：Lighting，地区：United States）
2. 确认任务状态变为 running
3. 等待任务完成
4. 确认任务状态变为 completed
5. 查看 b2b_leads 表有新数据
6. 通过 API 获取线索列表
7. 确认线索包含邮箱信息
8. 导出 Excel 并验证内容
```

## 5. 测试执行命令

```bash
# 运行所有测试
cd backend
pytest

# 运行指定模块
pytest app/tests/test_auth.py
pytest app/tests/test_social_tasks.py
pytest app/tests/test_b2b_tasks.py

# 运行并查看覆盖率
pytest --cov=app --cov-report=html

# 只运行 AI 相关测试
pytest -k "ai_"
```
